# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-03

### Added

- Initial release
- Multi-turn research agent with code generation
- Web search via Exa MCP (free) and Brave Search API
- LLM integration via OpenAI-compatible API
- Persistent key-value file store
- Code sandbox execution
- Auto-fix loop for generated code errors
- Interactive and CLI modes
- Utility functions: deduplication, filtering, flattening, coverage summarization
